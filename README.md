A Websaite for Northeastern University's Board Game Club (NUAGE) :)

This website currently features three pages:

Home:
A picture of the club's banner and a welcome message with a brief summary of what the club does and what it's mission is. Planning to have it include photos from various club meetings.

About:
Contains information about the club's history and how it was founded. Displays detailed bios for each eBoard member.

Games:
Offers users the ability to search through a vast database of the club's 300+ games. Site visitors will be able to search for specific games or filter them by things like complexity, max player count, or genre. Additionally, this page offers the ability to filter games by their storage location, which serves as a tool for eBoard members so they don't need to remember where every single game belongs.
